# Init Agent

The **full** init instructions for `/init-dev-cycle` live in the LangGraph package as
**`init_graph/prompt.md`** (bundled next to the graph; always the same text).

After `install.sh`, you still have **`custom.md`** here for optional local notes.
The executable graph loads **`prompt.md`** — not this file — so the orchestrated flow stays consistent.

For prompt-only / skill-based init (no graph), see **`core/skills/init-dev-cycle/SKILL.md`**.
